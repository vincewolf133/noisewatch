## ✨ Features
- **Real-Time Monitoring**: Instant noise triggers tracking with sub-second latency.
- **Historical Analysis**: View 60-point history charts for every sensor.
- **Data Export**: Download full historical logs as CSV files.
- **Live Leaderboard**: Ranked view of the quietest and noisiest rooms.
- **Premium UI**: Dark mode design with glassmorphism and smooth animations.

## 🛠️ Tech Stack
- **Frontend**: React.js, Vite, Chart.js, Phosphor Icons.
- **Backend**: Firebase Realtime Database.
- **Hardware**: ESP8266 (C++ / Arduino IDE) + Digital Sound Sensors.

## 🚀 Quick Start
To get the dashboard running locally:
```bash
cd web_dashboard
npm install
npm run dev
```
